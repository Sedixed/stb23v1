Opérations supportées

  • GET : /

    Retour : HTML

    Détail : Page d'accueil de l'application.

  • GET : /help

    Retour : HTML

    Détail : Page d'aide (celle-ci).

  • GET : /stb23/resume/xml

    Retour : XML

    Détail : Liste des spécifications stockées.

  • GET : /stb23/resume

    Retour : HTML

    Détail : Liste des spécifications stockées.

  • GET : /stb23/xml/{id}

    Retour : XML

    Détail : Contenu complet de la spécification d'identifiant {id}.

  • GET : /stb23/html/{id}

    Retour : HTML

    Détail : Contenu complet de la spécification d'identifiant {id}.

  • POST : /stb23/insert

    Transmis : Flux XML décrivant la spécification stb23 à ajouter, conforme au schéma XSD.

    Retour : XML

    Opération : Insère la spécification en base de données si celle-ci est valide et non présente.

  • DELETE : /stb23/delete/{id}

    Retour : XML

    Opération : Suppression de la spécification dont l’identifiant est {id}.
