Supported operations

  • GET : /

    Result type : HTML

    Detail : Home page.

  • GET : /help

    Result type : HTML

    Detail : Help page (this one).

  • GET : /stb23/resume/xml

    Result type : XML

    Detail : Stocked STB list.

  • GET : /stb23/resume

    Result type : HTML

    Detail : Stocked STB list.

  • GET : /stb23/xml/{id}

    Result type : XML

    Detail : Whole STB content (the one having the id {id}).

  • GET : /stb23/html/{id}

    Result type : HTML

    Detail : Whole STB content (the one having the id {id}).

  • POST : /stb23/insert

    Given : XML stream describing the STB to add.

    Result type : XML

    Operation : Inserts the given STB in database if valid and not duplicate.

  • DELETE : /stb23/delete/{id}

    Result type : XML

    Operation : Deletes the STB of id {id}.
